Projects

Here is a list of brief descriptions of my personal/group projects. Information might be out-dated. Please check my resume for most recent descriptions.

  • Object Detector - YOLO (Jan. 2021 - Mar. 2021)

  • Sentiment Analysis (Aug. 2020 - Sep. 2020)

    Description: A Python script that predicts the binary category of emotion in a sentence.

    Languages/Softwares: Python, scikit-learn

    Details:

    • Cleaned data with NLTK library by eliminating punctuations and stop words and lemmatizing each word remaining.
    • Fitted model using Naïve Byes algorithm in the scikit-learn and LSTM model in TensorFlow.
    • Achieved accuracy of 80% in training and 76% in testing.

  • Animals Classification (Dec. 2020 - Jan. 2020)

    Description: A Python script that classifies images of animals into ‘Cat’ or ‘Dog’.

    Languages/Softwares: Python, TensorFlow

    Details:

    • Preprocessed images to arrays of integers with the imread function in the OpenCV library.
    • Trained neuron network having convolutional and pooling layers using TensorFlow/keras.
    • Optimized hyperparameters using grid search.
    • Visualized the optimization process with TensorBoard.

  • Stock Price Prediction (Dec. 2020 - Jan. 2020)

    Description: A Python script that predicts weekly closing prices in the following month.

    Details:

    • Applied transformations on time series data provided in TensorFlow package.
    • Trained neuron network with LSTM, Dropout, and Dense layers.
    • Evaluated my model using cross-validation and other techniques.

  • Camber Crawler 3000 Game (August 2020)

    Description: A course group project implementing a command-line game in C++.

    Languages/Softwares: C++

    Details:

    • Applied design patterns including the observer pattern.
    • Demonstrates solid understanding of class dependencies and inheritance in ‘Player’, ‘Enemy’, and other classes.
    • Automated tests with bash programs in Linux systems to test moving and attack behaviors of the player.
